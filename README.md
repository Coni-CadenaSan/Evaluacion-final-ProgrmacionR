# Evaluacion-final-ProgrmacionR
Análisis del impacto del trabajo doméstico y de cuidados no remunerado en el bienestar, con perspectiva de género. Se aplica regresión logística binaria (logit) con datos propios (n=42). Variables: género, hijes, empleo y escolaridad. Incluye cuatro gráficas. R | Universidad de Guanajuato.
